Google Sheets setup
This app queues transactions on your phone and syncs them to your Google Sheet via a Google Apps Script web app URL. It takes about 5 minutes to set up once.

Steps:
1. Open your Google Sheet
2. Go to Extensions → Apps Script
3. Paste the script below and click Deploy → New deployment
4. Choose Web app, set access to Anyone
5. Copy the web app URL and paste it here
Web App URL
Paste your deployed Apps Script URL above. Transactions in the queue will POST to this URL and be inserted into your Transactions sheet.
